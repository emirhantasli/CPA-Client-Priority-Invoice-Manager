# CPA Client Priority & Invoice Manager

## Project Overview

CPA Client Priority & Invoice Manager is a Java Swing desktop application developed for the SEN2212 Data Structures and Algorithms II course.

The application helps Certified Public Accountants (CPAs) manage company records, invoices, and tax-related tasks through a user-friendly graphical interface.

## Features

* Add, search, and delete companies
* Create and manage invoices
* Add and prioritize tax tasks
* Save and load data from files
* Undo recent operations
* User-friendly Java Swing GUI

## Data Structures Used

* HashMap – Company management
* ArrayList – Invoice storage
* PriorityQueue – Tax task prioritization
* Stack – Undo operations

## Technologies

* Java 23
* Java Swing
* Eclipse IDE
* Java Collections Framework
* File I/O Operations
